# Superstore Sales Analysis (Excel Project)

## Overview
End-to-end Excel analysis of the Superstore dataset (9,994 transactions, 2014-2017) 
covering data cleaning, pivot tables, dashboard, and business insights.

## Tools Used
Excel (Pivot Tables, Data Cleaning, Charts, Dashboard Design)

## Process
1. Raw data import and cleaning (duplicate check, header fix, blank value check)
2. Built 4 pivot tables: Category, Region, Year-wise Trend, Top 10 Customers
3. Designed an interactive dashboard combining all 4 visualizations
4. Extracted 5 key business insights from the data

## Key Insights
1. Technology drives profit, not Furniture — despite lower sales, Technology 
   generated the highest profit (₹1,45,455).
2. West region is the strongest performer, contributing ₹1,08,418 in profit.
3. Consistent year-on-year growth from ₹4.8L (2014) to ₹7.3L (2017) in sales.
4. Top 10 customers contribute ₹1.54L (~6.7%) of total revenue.
5. Furniture needs a margin review — high sales but weakest profit among categories.
